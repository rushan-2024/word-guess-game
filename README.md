A fun and interactive Word Guessing Game built using Python and Tkinter. The game challenges players to guess a randomly selected word one letter at a time within a limited number of attempts. It features multiple difficulty levels, a hint system, restart functionality, and real-time score tracking, making it both engaging and beginner-friendly.
The project demonstrates core Python concepts such as GUI development, event handling, game state management, and input validation. It is ideal as a mini-project, learning exercise, or portfolio addition.
